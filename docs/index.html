포트폴리오

Dynamic Beat

Dynamic Beat는 자바(Java)로 개발한 2D 리듬게임입니다. 2D 게임을 개발했을 뿐 아니라 게임을 만드는 전 과정을 유튜브 강의로 촬영하여 무료로 배포하는 과정을 통해 많은 것을 배운 프로젝트입니다.

다락리타임 어플리케이션

한국교원대학교 학생들을 위해 개발한 무료 수강신청 도우미 안드로이드 앱입니다. 회원가입, 로그인, 강의 신청, 강의 시간표 보기, 강의 분석, 경쟁률 분석 등의 기능을 포함한 학생 자율 어플리케이션입니다.

MIDAS-Cook

웹 솔루션 공모전 수상작입니다. 학교 내 식단 알림, 이메일 전송, 공지사항, 자유게시판, 통계 다이어그램 등의 기능을 포함한 신개념 학생 식단 관리 웹 솔루션입니다.

다락리타임 웹 사이트

기존에 존재하던 다락리타임 어플리케이션을 반응형 웹 기술을 이용해 개편한 웹 사이트입니다. 마찬가지로 회원가입, 로그인, 강의 신청, 강의 시간표 보기 등의 기능을 제공합니다. 태블릿, 모바일, 노트북 등의 멀티 플랫폼을 지원합니다.
